── 使い方 ──

  • 🖌️色を選んでキャンバスに描く
  • 🎲 色ランダム (R)：ランダムな色に切り替え
  • ◻️ 消しゴム：Eキーで消しゴムとブラシ切り替え
  • ▶️ 再生：左上から１列ずつ音を鳴らす
  • ⏱️ 速さ：再生間隔（秒／ピクセル）を調整
  • 💾 画像保存／読み込み：描いた絵をまるごと保存・読込可

── 色→音 変換ルール ──

  1. 色相 (H) → 音階（周波数の基礎）
  2. 明度 (L) → オクターブ（高い or 低い音域）
  3. 彩度 (S) → 音色の「こもり／鮮明さ」
    • 低Ｓ → ノイズ多めで“こもった”音
    • 高Ｓ → 倍音フィルターを開放して“キラキラ”音
  4. 透明ピクセル → 無音エリア
